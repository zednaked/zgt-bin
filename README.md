# ZGT - ZeDs Godot Terminal

A Godot 4 GDExtension that provides a built-in PTY terminal emulator as an editor bottom-panel control. Works on X11 and Wayland. Linux only.

## Usage

1. Copy `addons/`, `bin/`, and `zgt.gdextension` into your Godot project root
2. Enable **ZeDs Godot Terminal** plugin in Project Settings → Plugins
3. Restart the editor — the **ZGT** tab appears in the bottom panel

## Build requirements

- Godot 4.2+
- Linux
- C++17 toolchain
- `godot-cpp` 4.3
